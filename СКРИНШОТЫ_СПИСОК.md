# Список скриншотов для курсовой работы FitTime

Сохраняй скриншоты в папку `Колясик/screenshots/`
Имена файлов указаны — используй их точно для автоматической вставки в docx.

---

## Обязательные скриншоты (16 штук):

### Авторизация
1. **login.png** — Окно входа в систему (LoginWindow). Заполненные поля логин/пароль, кнопка «Войти».

### Панель управления (Dashboard)
2. **dashboard.png** — Главная панель: 4 KPI-карточки сверху, график посещаемости, занятия сегодня, продажи, истекающие абонементы.

### Клиенты
3. **clients_list.png** — Список клиентов с данными в таблице. Видны фильтры (Все/Активные/Просроченные/Без аб.), поиск, кнопка «+ Добавить».
4. **client_profile.png** — Профиль конкретного клиента. Вкладка «Личные данные» с заполненными полями (ФИО, телефон, email, заметки).

### Абонементы
5. **membership_types.png** — Справочник типов абонементов: таблица с колонками Название, Дни, Кол-во посещений, Безлимит, Цена.
6. **membership_dialog.png** — Диалог «Оформить абонемент»: выбранный клиент, выбранный тип, даты начала/окончания, цена.

### Расписание
7. **schedule_week.png** — Расписание в режиме «Неделя»: календарная сетка с цветными блоками занятий (разные типы — разные цвета).
8. **class_card.png** — Карточка занятия: тип, тренер, дата, время, зал, макс. участников, статус, кол-во записанных.

### Посещения
9. **attendance.png** — Журнал посещаемости: фильтры по дате и клиенту, таблица со статусами (Отмечено/Не явился/Отменено).

### Тренеры
10. **trainers.png** — Список тренеров: ФИО, телефон, email, специализация, занятий/мес.
11. **trainer_dialog.png** — Диалог «Добавить тренера»: заполненные поля + привязка к аккаунту пользователя.

### Отчёты
12. **reports_attendance.png** — Вкладка «Посещаемость»: график по дням (столбчатая диаграмма), блок KPI справа, таблица «Топ занятий» снизу.
13. **reports_sales.png** — Вкладка «Продажи»: KPI-карточки (продано, выручка, средняя цена) + таблица продаж по типам.

### Пользователи (вход как Администратор!)
14. **users.png** — Список пользователей: логин, ФИО, роль (цветной бейдж), статус (Активен/Заблокирован), дата создания.
15. **user_dialog.png** — Диалог «Добавить пользователя»: логин, ФИО, роль (выпадающий список), временный пароль.

### Навигация
16. **nav_open.png** — Главное окно с раскрытой навигацией: все табы видны (Панель, Клиенты, Абонементы, Расписание, Тренеры, Посещения, Отчёты, Пользователи).

---

## Рекомендации:
- Входи под аккаунтом **Администратора** — тогда видны все разделы
- Заполни данными все экраны (не пустые таблицы)
- Разрешение: желательно Full HD (1920x1080) или выше
- Формат: PNG
